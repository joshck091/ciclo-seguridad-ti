## 🔄 Las 6 Fases del Ciclo

### 1️⃣ **Identificación**
```
🔍 Inventario de activos, clasificación y evaluación de riesgos
```
- Descubrimiento de activos críticos
- Clasificación de datos sensibles
- Mapeo de amenazas y vulnerabilidades
- **Responsable:** CISO / Responsable TI

### 2️⃣ **Protección**
```
🛡️ Controles técnicos y organizativos para mitigar riesgos
```
- Implementación de firewalls y MFA
- Políticas de acceso y cifrado
- Programas de formación
- **Responsable:** Equipo de Seguridad / SOC

### 3️⃣ **Detección**
```
🚨 Monitorización y alertas para eventos anómalos
```
- SIEM y EDR
- Correlación de eventos
- Análisis de comportamiento
- **Responsable:** SOC / Equipo OPS

### 4️⃣ **Respuesta**
```
⚡ Contención, análisis forense y erradicación del incidente
```
- Playbooks de respuesta
- Análisis forense
- Contención y comunicación
- **Responsable:** Equipo de Respuesta a Incidentes

### 5️⃣ **Recuperación**
```
🔧 Restauración de servicios y verificación de integridad
```
- Restauración desde backups
- Validación post-recuperación
- Plan de continuidad (DRP)
- **Responsable:** Equipo TI / Operaciones

### 6️⃣ **Evaluación & Mejora**
```
📈 Revisión de incidentes, métricas y planificación de mejoras
```
- Lecciones aprendidas (post-mortem)
- KPIs (MTTR, MTTD)
- Actualización de controles
- **Responsable:** CISO / Gobernanza

## 🚀 Instalación y Uso

### **Método 1: Uso Directo** (Recomendado)
```bash
# 1. Visita la demo en vivo
https://TU-USUARIO.github.io/ciclo-seguridad-ti/

# 2. Usa el botón "Descargar HTML" integrado en la página
```

### **Método 2: Clonar Repositorio**
```bash
# 1. Clonar el repositorio
git clone https://github.com/TU-USUARIO/ciclo-seguridad-ti.git

# 2. Navegar al directorio
cd ciclo-seguridad-ti

# 3. Abrir index.html en tu navegador
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

### **Método 3: Descargar ZIP**
```bash
# 1. Click en el botón verde "Code" → "Download ZIP"
# 2. Extraer el archivo ZIP
# 3. Abrir index.html con cualquier navegador moderno
```
